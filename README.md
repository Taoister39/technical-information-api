## docker 使用方式


1. 更改web服务的dockerfile位置，确保相同
2. 在本项目下进入`powershell`
3. 运行`docker compose build`
4. 运行`docker compose create`，创建compose镜像
5. 运行`docker compose start`，使得docker实例化容器
